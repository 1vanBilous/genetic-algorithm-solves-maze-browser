This application solves 2D mazes using genetic algorithm under the hood with visualization of the process. Mazes are built by user and can be saved into browser local storage for future use.
Controls:
  • Left mouse button click - build wall
  • Mouse wheel click - set start
  • Right mouse button click - set finish
  • CTRL+S or UI corresponding button - save template
You can set desired algorithm parameters in corresponding form in the left-hand-side. Feel free to play with parameters to see how the algorithm behaves!
Source code can be found on GitHub, in case you want to explore it.
